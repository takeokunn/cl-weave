# Support

Use the [Support Policy](https://takeokunn.github.io/cl-weave/support-policy.html)
for supported environments, scope boundaries, and the information to include in
a request.

- Report reproducible defects, documentation gaps, and concrete feature
  requests through
  [GitHub Issues](https://github.com/takeokunn/cl-weave/issues/new/choose).
- Send fixes that can be described and validated locally as pull requests; see
  [CONTRIBUTING.md](CONTRIBUTING.md).
- Report security vulnerabilities privately through
  [GitHub Security Advisories](https://github.com/takeokunn/cl-weave/security/advisories/new).

Do not include vulnerability details in public issues or discussions.
